#  Student Performance Prediction System

A Machine Learning project that predicts student performance (Low, Medium, High) based on academic and behavioral factors using Python and Streamlit.

---

##  Project Overview

This project aims to analyze student data and predict their academic performance using machine learning techniques. It includes data preprocessing, model training, and a user-friendly web interface for real-time predictions.

---

##  Features

- Predicts student performance (Low / Medium / High)
- Uses academic inputs like:
  - G1 (First Test Score)
  - G2 (Second Test Score)
  - Study Time
  - Absences
- Interactive web app built using Streamlit
- Real-time predictions with simple UI

---

##  Tech Stack

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit

---

##  Machine Learning Workflow

1. Data Collection (real-world dataset)
2. Data Cleaning & Preprocessing
3. Feature Engineering
4. Model Training (Classification)
5. Model Evaluation
6. Deployment using Streamlit
